## What Are Design Patterns?  
Design Patterns are proven solutions to common software design problems. They are categorized into three types:  
- **Creational Patterns**: Focus on object creation mechanisms.  
- **Behavioral Patterns**: Deal with object interaction and responsibility delegation.  
- **Structural Patterns**: Simplify the design by focusing on class and object composition.
